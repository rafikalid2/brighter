Documentation :

To use the brighter chart API you will need to include the jquey, than add the following lines to your code

<link rel="stylesheet" href="brighter-chart.css"> <script src="brighter-chart.js"></script>

to create the serial chart, create a new object of serial class

var serial = new Serial();

the api need a container so specify a container for the chart by the adding the next line

serial.container('#chart');

the api is based on build design pattern so all functions return the current object

serial.container('#chart') .data(data) .style({offsetRight: 100});

the data() function take as argument an array of data that you want to draw, the data need to be in this form

data = [ { "date": "01", "distance": 227, "latitude": 280, "duration": 408 }, { "date": "02", "distance": 371, "latitude": 230, "duration": 482 }, { "date": "03", "distance": 433, "latitude": 300, "duration": 562 } ];

by the style() function you can change the chart style, you can change

+ offsetTop is the space between the chart and the container top, by default it take 10px
+ offsetRight the space between the chart and the container right, by defualt it is 80px
+ offsetBottom the space between the chart and the container bottom, by defualt it is 80px
+ offsetLeft the space between the chart and the container left, by defualt it is 80px

serial.container('#chart') .data(data) .style({offsetRight: 100}) .addAxis({field:"date", type:"horizontal"}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00", offsetTitle: 10}) .addAxis({field:"latitude", type:"vertical", position:'right', display:false, stepsNumber: 11, offset: 70}) .addAxis({field:"duration", type:"vertical"});

to define an axis use addAxis() function, this function take as argument an object of options which define the axis
the options as following :

+ field the value key in array data specified for this axis
+ type axis type the horizontal one or vertical
+ position the position is only work if the axis is vertical so you can choose between left or right, by default it is left
+ color the color of the axis and steps, #b7b7b7 is the default value
+ textColor the color of the axis steps values, by default it is black color #000
+ offset the offset value, by this attribute you can change the position of values steps, default value is 0
+ offsetTitle the space between the axis and its title, by default this value is 0
+ titleColor the axis title color, default value is black color #000
+ display by default this option is true, if you don't want to show the axis so chage it to false
+ stepsNumber number of values to show in axis

serial.container('#chart') .data(data) .style({offsetRight: 100}) .addAxis({field:"date", type:"horizontal", display:false}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00"}) .addAxis({field:"latitude", type:"vertical", position: "right", display:false, stepsNumber: 11, offset: 45}) .addAxis({field:"duration", type:"vertical"}) .addColumn({field: "distance", position: '5 30', offset: 5}) .addColumn({field: "latitude", position: '35 30'}) .addColumn({field: "duration", position: '65 30'});

the serial chart support two types of graph the column, and line graph, to add an column graph use the addColumn() function,
this function take an object of options which define the graph, this options as following

+ field the value key in array data specified for this graph
+ color the color of the graph, in case is not defined, it will take an random color
+ border the border color, by default it is #71bcde
+ opacity the opcaity of the graph, by default it is 1
+ position the first value is where the graph will start, and second one is where the graph will end, all those value is in percentage, default value is "5 90"
+ offset is the margin left and right of the graph, by defult it is 0
+ class to specify an class for all the graph, by default it is "colunms", the graph parts also has an fixed class "column"

serial.container('#chart') .data(data) .style({offsetRight: 140}) .addAxis({field:"date", type:"horizontal"}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00", offsetTitle: 10}) .addAxis({field:"latitude", type:"vertical", position:'right', display:false, stepsNumber: 11, offset: 70}) .addAxis({field:"duration", type:"vertical"}) .addColumn({field: "distance", position: '5 30', offset: 5}) .addColumn({field: "latitude", position: '35 30'}) .addColumn({field: "duration", position: '65 30'}) .addLine({field: "duration", animate: "dash-line", class: "animate"}) .addLine({field: "distance", joint: "circle", animate: "draw-line", class: "animate dis"}) .addLine({field: "latitude", joint: "circle"});

the serial chart support two types of graph the column, and line graph, to add an column graph use the addColumn() function,
this function take an object of options which define the graph, this options as following

+ field the value key in array data specified for this graph
+ color the color of the graph, in case is not defined, it will take an random color
+ opacity the opcaity of the graph, by default it is 1
+ strokeWidth this option define the size of the line, by default it is 2
+ animate this specify an animation for the line, there is two animations "draw-line" and "dash-line" animation, to make this animations works you need to add animate class to the group by using the class option
+ joint by this options you can change the joint types, the chart propose two types rect or circle joint, by defualt it take "rect" value
+ class to specify an class for all the graph, by default it is "lines", the graph is one path and has "line" class, the joint also inder an group with class name "points", the joint it self has "point" class

After finishing the chart configuration you need to build it, so use the build() function as showing next

serial.container('#chart') .data(data) .style({offsetRight: 140}) .addAxis({field:"date", type:"horizontal"}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00", offsetTitle: 10}) .addAxis({field:"latitude", type:"vertical", position:'right', display:false, stepsNumber: 11, offset: 70}) .addAxis({field:"duration", type:"vertical"}) .addColumn({field: "distance", position: '5 30', offset: 5}) .addColumn({field: "latitude", position: '35 30'}) .addColumn({field: "duration", position: '65 30'}) .addLine({field: "duration", animate: "dash-line", class: "animate"}) .addLine({field: "distance", joint: "circle", animate: "draw-line", class: "animate dis"}) .addLine({field: "latitude", joint: "circle"}) .build();

To delete the chart you can use the delete() function as following

serial.delete();

To make the chart responsive add the following code lines

$( window ).resize(function() { serial.delete(); serial.build(); });

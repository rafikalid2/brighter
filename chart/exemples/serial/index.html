Documentation :

To use the brighter chart API you will need to include the jquey and the jquery-svg API, than add the following lines to your code

<link rel="stylesheet" href="brighter-chart.css"> <script src="brighter-chart.js"></script>

to create the serial chart, create a new object of serial class

var serial = new Serial();

the api need a container so specify a container for the chart by the adding the next line

serial.container('#chart');

the api is based on build design pattern so all functions return the current object

serial.container('#chart') .data(data) .style({offsetRight: 100});

the data() function take as argument an array of data that you want to draw, the data need to be in this form

data = [ { "date": "01", "distance": 227, "latitude": 280, "duration": 408 }, { "date": "02", "distance": 371, "latitude": 230, "duration": 482 }, { "date": "03", "distance": 433, "latitude": 300, "duration": 562 } ];

by the style() function you can change the chart style, you can change the offsetTop, offsetRight, offsetBottom and offsetLeft values

serial.container('#chart') .data(data) .style({offsetRight: 100}) .addAxis({field:"date", type:"horizontal", display:false}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00"}) .addAxis({field:"latitude", type:"vertical", position: "right", display:false, stepsNumber: 11, offset: 45}) .addAxis({field:"duration", type:"vertical", display:false});

to define an axis use addAxis() function, this function take as argument an object of options which define the axis
the options as following :

+ field the value key in array data specified for this axis
+ type axis type the horizontal one or vertical
+ position the position is only work if the axis is vertical so you can choose between left or right
+ color the color of the axis and steps
+ textColor the color of the axis steps values
+ offset the values offset, by this attribute you can change the position of values steps
+ display by default this option is true, if you don't want to show the axis so chage it to false
+ stepsNumber number of values to show in axis

serial.container('#chart') .data(data) .style({offsetRight: 100}) .addAxis({field:"date", type:"horizontal", display:false}) .addAxis({field:"distance", type:"vertical", position: "right", titleColor: "#f00"}) .addAxis({field:"latitude", type:"vertical", position: "right", display:false, stepsNumber: 11, offset: 45}) .addAxis({field:"duration", type:"vertical"}) .addColumn({field: "distance", offset: 5}) .addColumn({field: "latitude", offset: 10}) .addLine({field: "duration", animate: "dash-line", class: "animate"}) .addLine({field: "distance", joint: "circle", animate: "draw-line", class: "animate dis"});
